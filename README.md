# 🎬 MovieSearch - Movie Discovery Application

A powerful and elegant movie search application that allows users to search for movies, TV series, and episodes using the **OMDb API**. Get detailed information about any movie including ratings, cast, plot, and more.

## 📖 About The Project

MovieSearch is a web-based application that helps users discover information about their favorite movies and TV shows. Simply type any movie title, and the app will fetch real-time data from the OMDb API, displaying comprehensive information including plot summaries, cast details, IMDB ratings, and more.

### Why I Built This

I created MovieSearch to demonstrate my ability to:
- Integrate third-party REST APIs
- Handle asynchronous JavaScript operations
- Create responsive, user-friendly interfaces
- Build functional applications with real-world data

## ✨ Features

- 🔍 **Search Movies** - Find any movie, TV series, or episode by title
- 🎯 **Filter Results** - Filter by movie, series, or episode type
- 📊 **Detailed Information** - View complete movie details including:
  - Cast and crew members
  - IMDB ratings (out of 10)
  - Release date and runtime
  - Genre and language
  - Full plot summary
- 🔗 **IMDB Integration** - Direct links to official IMDB pages
- 📱 **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- 🚀 **Load More Results** - Pagination support for large result sets
- ⚡ **Fast Performance** - Optimized API calls with loading states
